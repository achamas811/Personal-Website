Projects

    Empowering Data

    Description

    This project explores the relationship between household size and the average number of years for female schooling in Colombia and Peru. The data used is from the "livwell135.csv" file. The data used for analysis is from 2000 - 2015. By analyzing this data, I was able to answer the following research question: How do geographical, social, and economic events impact a young woman's ability to continue in school?

    Technologies Used

    • Python (Matplotlib, pandas)
    • Command Prompt/Terminal
    • Microsoft Visual Studio Code
    • Git and GitHub

    What Did I Learn?

    • I learned how to use Python and some of its libraries
    • The results from this project show that there is a correlation between household size and the average number of years for female schooling in Colombia and Peru.
      • Disclaimer: A correlation does not imply causation!
